## Hamtoon - 웹툰 사이트 만들기

- 참고 : [플러터로 웹툰 앱 만들기 노마드 코더](https://nomadcoders.co/flutter-for-beginners)
- 플러터로 제작된 강의인데 난 플러터 모르고 근데 만들어보고 싶어서 API만 쏙 빼옴

### 사용한 프로그래밍 언어

- Python - 3.12
- Typescript - 버전 모름ㄴ아ㅓ런래저

### 사용한 라이브러리

- Vite - 바닐라 + 부트스트랩
- Flask - 얘도 버전 몰ㅇ라아ㅣㅓ
- sqlite - DB 저장하려고
